Educational Walkthrough

What is Machine Learning?

Machine Learning (ML) is a branch of Artificial Intelligence that allows computer models to automatically learn patterns from data and make predictions on unseen info, without being explicitly programmed for that specific task.

What is Classification?

Classification is a supervised learning task where the model's goal is to categorize an input item into one of several predefined classes. Here, the model uses 4 physical measurements to predict one of 3 Iris species labels.

Why K-Nearest Neighbors?

During Python training, multiple models (SVC, Random Forest, Logistic Regression, Decision Tree, KNN) were trained. The K-Nearest Neighbors (KNN) algorithm achieved the highest cross-validation and test accuracy (100%), making it the select champion model.

Why Browser Inference?

Normally, machine learning predictions require sending user data to cloud servers. By converting the model to **ONNX format** and using **ONNX Runtime Web**, we load the model directly in JavaScript. Predictions are fast, offline-ready, and respect privacy.
